# 🛋 配套工具

规范 Lint 工具 NPM 包
JavaScript 编码规范
TypeScript 编码规范
Node 编码规范		 ESLint (opens new window) eslint-config-re (opens new window)

Git 规范 commitlint (opens new window) commitlint-config-re (opens new window)
